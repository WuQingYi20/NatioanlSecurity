RL Formulation: KG Traversal as MDP

How knowledge graph investigation maps to a Markov Decision Process.

S — State Space (48-dim)

Agent observes a fixed-size vector encoding its current position, neighborhood, and investigation progress.

A — Action Space

MultiDiscrete([3, 20]) — action type + parameter

🔍

explore_node(neighbor_idx)

Move to a visible neighbor. Must be within semantic window scope. Collects evidence.

🔓

request_clearance(level)

Ask human to widen semantic window. Costly: -0.05 if granted, -0.10 if denied. 80% approval rate.

📦

submit_evidence_bundle()

End episode. Reward based on F1 of threat nodes in collected evidence vs ground truth cluster.

R — Reward Function

R = α·Threat_Confidence + β·Efficiency − γ·Privacy_Penalty
α = 0.3
Weak signal
Based on biased historical labels. Per-step: +0.06 for threat node. Submission: α·F1·10
β = 0.1
Noisy but objective
Per-step cost: -0.001. Submission bonus: β·(1 - steps/budget)·5
γ = 1.0
Strong, objective signal
Visiting benign node: -0.3. Low-confidence link: -0.1. Unvisited evidence: -2.0 each
Key insight: γ >> α — privacy penalty dominates. The agent must learn to be precise rather than thorough. Visiting innocent citizens is heavily penalised. This encodes the constitutional constraint directly into the learning signal.

T — Transition Dynamics

Deterministic component: explore_node moves agent to selected neighbor.

Stochastic components:
• request_clearance: 80% grant / 20% deny
• Episode start: 70% at threat seed, 30% random node
• Entity resolution noise on ~15% of cross-agency links

Constraints (guardrails baked into env):
• Read-only — agent cannot modify graph
• Semantic window — only allowed edges visible
• Budget — max 50 steps per episode

Three-Layer Guardrail Architecture

Training-time (reward shaping) + Runtime automated (G1-G6) + Human-in-the-Loop. Click each for rationale.

Layer 1: Training-Time (baked into reward)

γ >> α Privacy penalty dominates

Visiting benign nodes costs 10x more than the signal from finding threats.

Constitutional constraint: mass surveillance is illegal. Agent learns precision over recall.

Low-confidence link penalty

Traversing edges with ER confidence < 0.6 incurs additional -0.1 penalty.

Prevents agent from acting on uncertain entity matches — wrong person is worse than no lead.

Budget constraint (50 steps)

Episode truncates after 50 steps with -0.5 penalty.

Forces efficient investigation. Unbounded traversal = unbounded privacy intrusion.

Layer 2: Runtime Automated (G1-G6)

G1: Confidence Calibration

Platt scaling ensures agent's confidence scores map to actual probabilities.

Raw neural network logits are poorly calibrated. A "90% confident" agent should be right 90% of the time.

G2: Scope Boundary Check

Every node access validated against investigation mandate.

Prevents scope creep. Investigation into company X cannot silently expand to company Y.

G3: Anti-Hallucination

Every evidence item verified against actual database entry.

LLMs can fabricate plausible-sounding entity IDs. A fake personnummer in an evidence bundle = wrongful investigation.

G4: Evidence Sufficiency

Minimum evidence threshold before bundle accepted.

Prevents empty or trivial submissions. Investigation must have substance.

G5: Disproportion Detector

Bias check on evidence demographics and patterns.

If 80% of flagged individuals share a protected characteristic, something is wrong with the model or data.

G6: Independent Risk Assessment

System computes risk score independently. Agent's self-reported confidence discarded.

Agent has incentive to overstate confidence. Independent assessment prevents self-serving risk evaluation.

Layer 3: Human-in-the-Loop

Evidence-First Presentation

Agent's conclusion hidden until operator forms independent assessment.

Prevents anchoring bias. If you see "THREAT: 95%" first, you evaluate evidence to confirm, not challenge.

Mandatory Alternative Hypothesis

Operator must articulate a benign explanation before approving.

Forces adversarial thinking. "Could this pattern be legitimate?" must be answered.

Risk-Tiered Routing

Low → standard analyst | Medium → senior + reasoning | High → + delay | Critical → dual approval

Proportionality: a tax audit flag needs less scrutiny than a counter-terrorism flag.

Anti-Rubber-Stamp

System monitors operator approval rates. Unusually high approval rates flagged.

Human oversight is worthless if the human always clicks "approve". Deliberation time enforced for High+ risk.
